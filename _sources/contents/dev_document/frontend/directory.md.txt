# ディレクトリ構成

```
├─api				...API実行時に必要なデータ定義
├─component			...すべてのコンポーネント
│  ├─atoms			...最小単位のコンポーネント
│  │  ├─formParts		...フォーム関連パーツ
│  │  └─security		...認証認可関連パーツ
│  ├─mainContents		...あるページのmainコンテンツとなるコンポーネント群
│  ├─molecules		...最小単位のコンポーネントの集まり
│  │  └─form		...フォーム関連パーツ
│  └─organisms		...一定の機能を成すコンポーネントの集まり
├─pages			...1ページあたりの表示パーツを集約するためのコンポーネント
└─utility			...ユーティリティ関数など
```
